Why Wi-Fi Security Matters

Your Router is the Front Door

The router is often the only thing standing between your home network and the internet. A weak router configuration puts every connected device at risk.

Think of it like this...

An unsecured router is like leaving your front door unlocked with the key still in it — anyone nearby could walk straight onto your network.

WPA2 vs WPA3

WPA2

  • Standard since 2004, widely supported
  • Uses AES encryption
  • Vulnerable to offline password-guessing attacks
  • Still common on older routers/devices

WPA3

  • Newer standard with stronger protections
  • Individualised encryption per device
  • Resists offline password-guessing attacks
  • Use whenever your router and devices support it

The Default Credentials Problem

Every router model ships with a known default admin username and password — published online for anyone to find.

SSID Naming & Guest Networks

Rename the SSID

Avoid names that reveal the router model or your address

Hide the SSID

Optional extra step — stops it appearing in casual scans

Guest Network

Keeps visitors off your main network and devices

Good to Know

Hiding an SSID is a minor deterrent, not real security — a strong password and WPA3 matter far more.

Router Admin Hardening Checklist

SettingRecommended Action
Admin passwordChange from default immediately
Wi-Fi securitySet to WPA3 (or WPA2 if unavailable)
FirmwareKeep updated for security patches
Remote admin accessDisable unless absolutely needed
WPS (push-button pairing)Turn off — a known weak point

Practical Task

Harden a Router Admin Page

  • Using a spare router or admin-panel screenshots, log in to the admin page
  • Change the default admin password to something strong
  • Set the Wi-Fi security mode to WPA3 (or best available)
  • Rename the SSID and, optionally, hide it
  • Set up a separate guest network for visitors

Stay Safe

Only practise on a spare/training router or approved screenshots — never change settings on a live network without permission.

Q1: Compared to WPA2, WPA3 offers...

AWeaker encryption
BStronger, more resistant encryption
CNo password requirement
DFaster internet speeds only
Answer: B - WPA3 improves encryption and resists offline password-guessing attacks.

Q2: Why should you change a router's default admin password?

ADefault passwords are illegal
BDefault passwords are publicly known for each model
CIt makes Wi-Fi faster
DIt's required to connect any device
Answer: B - Default credentials are published online, making them trivial for attackers to find.

Q3: What is a guest network for?

AMaking the router run faster
BKeeping visitor devices separate from your main network
CReplacing the admin password
DHiding your router from the internet entirely
Answer: B - A guest network lets visitors get online without accessing your trusted devices and files.

Key Takeaways

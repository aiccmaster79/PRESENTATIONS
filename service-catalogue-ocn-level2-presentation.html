Service Catalogue

Build a mini IT service catalogue

OCN NI Level 2–3 — IT support / service delivery

Learning Objectives

By the end of this session, you will be able to:

  • Explain what an IT service catalogue is for
  • List services users can request (and what is out of scope)
  • Write clear service entries with owner and request path
  • Link catalogue items to tickets and expectations
  • Build a mini catalogue for a school/college scenario

Core Ideas

Catalogue purpose

  • Show what IT offers
  • Reduce mystery requests
  • Set expectations
  • Point to how to request

Entry fields

  • Service name
  • Who it is for
  • How to request
  • Owner / hours

Demo Sequence

  1. Brainstorm services already offered in the school
  2. Pick 6–8 services for a mini catalogue
  3. Write one complete entry on the projector
  4. Map each entry to a ticket type/category
  5. Mark two items as out of scope with alternatives
  6. Publish the mini catalogue to the class share

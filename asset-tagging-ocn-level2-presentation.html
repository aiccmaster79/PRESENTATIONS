Asset Tagging

Follow a device through an asset register

OCN NI Level 2–3 — IT support / service delivery

Learning Objectives

By the end of this session, you will be able to:

  • Explain why organisations tag and track IT assets
  • Record a device in an asset register
  • Apply a physical asset tag/label
  • Update status through lifecycle (in stock → issued → retired)
  • Link assets to users, tickets, and warranty info

Core Ideas

Why track assets

  • Know what you own
  • Support and warranty
  • Theft/loss control
  • Audit and funding evidence

Register fields

  • Asset tag ID
  • Make/model/serial
  • Location / user
  • Status / purchase date

Demo Sequence

  1. Show an untagged laptop and blank register row
  2. Generate/assign an asset ID and print/apply a label
  3. Enter make, model, serial, location, status
  4. Issue the device to a user and update the register
  5. Log a sample ticket referencing the asset ID
  6. Retire/dispose status change at end of life

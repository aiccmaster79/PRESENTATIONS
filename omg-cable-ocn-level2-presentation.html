Ethics & Legal First

An O.MG Cable is a commercially sold red-team tool. Deploying one against any device or organisation without a signed rules-of-engagement document is unauthorised computer misuse and can be prosecuted.

Classroom rule

We study the engineering and detection of implant cables to build defensive skill. Any hands-on use is confined to your own isolated lab equipment — never a shared or production device.

What is an O.MG Cable?

It looks and functions exactly like a genuine USB/Lightning charge-and-sync cable. Hidden inside the connector housing is a tiny microcontroller with its own Wi-Fi radio, capable of emulating a keyboard (HID) once plugged into a target computer.

Key point

The victim device still charges and transfers data normally — there is no visible sign anything is different, which is exactly what makes it useful in physical security testing.

How It Works

Looks Like a Normal Cable

Hidden Wi-Fi Implant

Operator's Phone / Laptop

  1. Cable is plugged into the target machine (via social engineering or physical access)
  2. The implant broadcasts its own Wi-Fi access point or joins a known network
  3. The operator connects to the cable's web-based control panel remotely
  4. Pre-written payloads (keystroke scripts) trigger on demand or via geofencing

Technical Capabilities

Keystroke Injection

Sends DuckyScript-style payloads as if typed on a real keyboard

Remote Wi-Fi Control

Operator triggers payloads from a phone within radio range

Geofencing

Can arm/disarm itself based on Wi-Fi signal proximity

Keystroke Logging

Some variants can log typed input for later retrieval

Self-Destruct / Forensics Resistance

Can wipe payloads if it detects analysis attempts

Fully Functional Cable

Still charges and transfers data — no operational giveaway

O.MG Cable vs Rubber Ducky

O.MG Cable

  • Disguised as an everyday charge cable
  • Wi-Fi remote control & geofencing
  • Stays plugged in — persistent presence
  • Harder to spot: fully working cable

Rubber Ducky

  • Disguised as a USB flash drive
  • Payload runs once, on insertion
  • Removed immediately after use
  • Easier to explain away if spotted

Detection & Defensive Controls

Cable inventory & provenance

Only use vendor-supplied cables; log serials for high-security areas

USB data blockers

"Charge-only" adaptors physically prevent data pins from connecting

Endpoint HID controls

Block or alert on new keyboard-class USB devices

Wireless monitoring

WIDS can flag unexpected rogue access points near secure areas

Physical security

Clean-desk policy, visitor escort, and CCTV reduce planting opportunities

Staff awareness

Never use a "found" or "gifted" cable — bring it to IT Security

Absolute Vs Relative References

Why some spreadsheet formulas move — and others stay fixed

OCN NI Level 2–3 — Data modelling / analytics

Learning Objectives

By the end of this session, you will be able to:

  • Contrast relative and absolute cell references
  • Use the $ lock (e.g. $B$2) appropriately
  • Predict what happens when a formula is filled down/across
  • Build a VAT or tax model that needs a fixed rate cell
  • Debug a broken fill caused by the wrong reference type

Formulas That Travel

When you copy a formula down a column, relative references shift with each row. Absolute references (with $) stay locked on one cell — perfect for a tax rate, constant, or lookup table corner.

Reference Types

TypeExampleWhen copied downUse for
RelativeA2Becomes A3, A4…Row-by-row calculations
Absolute$B$2Stays $B$2Rates, constants
MixedA$2 or $A2Locks row or columnGrids / matrices

Tip

Shortcut tip: F4 (Windows) / Cmd+T (Mac Excel) cycles reference types.

Demo Sequence

  1. Show a price list and a single VAT rate cell (e.g. 20% in $E$1)
  2. Write =B2*$E$1 for the first row — fill down — all rows use the rate
  3. Deliberately use =B2*E1 without $ — fill down and show broken/zero results
  4. Fix with absolute references and re-fill
  5. Optional: show mixed refs in a simple multiplication table

Test Your Knowledge!

Let's see what you've learned about cell references

Q1: In Excel, $C$5 means…

AColumn C only is locked
BBoth column C and row 5 stay fixed when copied
CThe cell is deleted
DIt is a currency format only
Answer: B — Full absolute reference — neither column nor row shifts.

Q2: Relative references are best when…

AEvery row must point at the same constant cell
BEach row should calculate from its own adjacent cells
CYou never copy formulas
DYou only print the sheet
Answer: B — Relative refs are designed to adapt as you fill down or across.

Q3: A VAT rate in one cell used by 200 rows should be…

ARelative, so it drifts
BAbsolute (or named), so every formula hits the same rate
CHidden in a chart title
DTyped as text in every row
Answer: B — One source of truth — lock it with $ or a named range.

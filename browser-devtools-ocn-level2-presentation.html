Learning Objectives

By the end of this session, you will be able to:

  • Open browser DevTools and find Elements, Console and Network
  • Inspect HTML structure and CSS rules for a selected element
  • Edit text/styles live to prototype a change
  • Read a simple Console message or error
  • Check Network status codes for a page load (overview)

Inspect Workflow

  1. Right-click → Inspect (or F12)
  2. Use the element picker to click a heading or button
  3. Watch the HTML highlight and the Styles pane update
  4. Toggle a CSS property (e.g. colour) — page updates live
  5. Remind: refresh loses unsaved DevTools edits — change source files to keep them

Demo Sequence

  1. Open the prepared demo site files in VS Code and in the browser
  2. Inspect a heading — show tag name, classes and computed colour
  3. Live-edit the heading text and a CSS colour/size
  4. Introduce a deliberate JS error; find it in the Console
  5. Open Network, reload, point out 200 vs 404 for a missing image
  6. Toggle device toolbar and note layout changes (teaser for responsive)

AThe page’s JavaScript hit a problem
BThe chair is too low
CGit is uninstalled

AAlways save to disk automatically

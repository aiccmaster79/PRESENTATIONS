Cleaning Messy Data

Clean a dataset before analysis

OCN NI Level 2–3 — Data modelling / analytics

Learning Objectives

By the end of this session, you will be able to:

  • Spot common data quality issues in a CSV
  • Fix duplicates, blanks, inconsistent spellings, and formats
  • Standardise dates, categories, and headers
  • Document cleaning steps for reproducibility
  • Compare dirty vs clean results in a simple summary

Core Ideas

Messy data looks like

  • Extra spaces / mixed case
  • Duplicates
  • Missing values
  • Broken dates/numbers

Cleaning moves

  • Trim and proper case
  • Remove or flag duplicates
  • Decide on blanks
  • Unify formats

Demo Sequence

  1. Open a prepared messy CSV in Excel/Power BI
  2. Profile the data: blanks, uniques, obvious errors
  3. Fix headers and data types
  4. Clean categories and duplicates with visible steps
  5. Save a cleaned copy (never overwrite the raw file)
  6. Show how a summary changes after cleaning

Practical Notes

Keep the raw file

Clean on a copy

Document steps

Someone must replay them

Blanks policy

Delete, impute, or flag

Do not invent facts

Be honest about gaps

Why Backups Matter

Hard drives fail, laptops get stolen, files get accidentally deleted, and ransomware locks entire systems. A backup is the difference between an inconvenience and a disaster.

Drives can fail with no warning

Human Error

Accidental deletes happen to everyone

Ransomware

Theft / Disaster

Fire, flood or theft can destroy a device instantly

The 3-2-1 Rule Explained

The golden rule of backups

A single backup copy is not enough. The 3-2-1 rule spreads your risk so no single failure can wipe out everything.

3 Copies

Your original data plus two backups

2 Media Types

e.g. internal SSD and an external drive

1 Offsite

A copy stored away from the original location

Backup Types: Full vs Incremental

Full Backup

  • Copies every single file each time
  • Slower and uses more storage
  • Simple, single-step restore
  • Good starting point / weekly baseline

Incremental Backup

  • Only copies files changed since last backup
  • Fast and storage-efficient
  • Restore needs the full backup + all increments
  • Good for frequent daily backups

Test Your Knowledge!

Let's see what you've learned about backups

Q1: In the 3-2-1 rule, what does the "2" refer to?

A2 users allowed access
B2 different storage media types
C2 minutes to complete a backup
D2 offsite locations
Answer: B — Using two different media types (e.g. internal drive and external drive) means one type of failure can't destroy every copy.

Q2: What's the main difference between a full and an incremental backup?

AIncremental only backs up changed files
BFull backups are always smaller
CThey are exactly the same thing
DIncremental backups can't be restored
Answer: A — Incremental backups only capture what's changed since the last backup, saving time and space.

Q3: What's the best defence against losing data to ransomware?

APaying the ransom immediately
BHaving offline/offsite backups to restore from
CTurning off the computer forever
DHoping it doesn't happen
Answer: B — A clean, disconnected backup lets you restore files without paying criminals or relying on their word.

What is Parallels Desktop?

Definition

Parallels Desktop is a commercial Type 2 hypervisor for Mac. It runs a full guest operating system — usually Windows — as an application window alongside macOS, without rebooting.

Think of it like this...

It's like having a second computer inside a window on your Mac desktop — you can drag files between them, copy-paste, and even run Windows apps side-by-side with Mac apps.

Type 2 Hypervisor Architecture

Guest OS (Windows) & Apps
Parallels Desktop (Hypervisor)
macOS (Host Operating System)
Mac Hardware

Why "Type 2"?

  • It installs on top of an existing OS (macOS), not directly on hardware
  • macOS handles the real hardware; Parallels asks macOS for CPU, memory and disk resources
  • Contrast with Type 1 hypervisors (e.g. VMware ESXi) which run directly on bare metal

Install & Configure — Step by Step

  1. Install Parallels Desktop — download from the Mac App Store or Parallels site
  2. Create a new VM — choose "Install Windows" or attach an existing ISO
  3. Allocate resources — set virtual CPU cores, RAM and disk size for the guest
  4. Complete guest install — Windows installs inside the new virtual disk
  5. Install Parallels Tools — adds shared clipboard, drag-and-drop, and better graphics
  6. Choose a view mode — full screen, windowed, or Coherence (Windows apps appear on the Mac dock)

Parallels vs Boot Camp

Parallels Desktop

  • Windows runs alongside macOS, no reboot
  • Easy snapshots and quick resets
  • Some performance overhead vs native
  • Great for testing, demos and light use
  • Paid annual licence

Boot Camp

  • Reboots into Windows natively
  • Full hardware performance (e.g. gaming)
  • No macOS running at the same time
  • Free, built into macOS (Intel Macs)
  • Slower to switch between OSes

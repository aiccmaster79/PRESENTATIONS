The Storage Landscape

Every file you save has to live somewhere. Choosing the right storage is a balance of speed, capacity, cost and how portable or safe you need it to be.

HDD

Spinning platters, cheap, high capacity

SSD

Flash memory, fast, no moving parts

Cloud

Stored on someone else's server, anywhere access

USB

Small, portable, quick to grab and go

Speed at a Glance

~150MB/s HDD
~550MB/s SATA SSD
3000+MB/s NVMe SSD
VariesCloud / USB (depends on link)

Why it matters

A faster drive means quicker boot times, snappier apps and shorter file-copy waits — but cloud and USB speeds also depend on your internet connection or USB port version.

Cloud vs Local Storage

Local

  • Instant access, no internet needed
  • One-off cost, you own the drive
  • Full control over your files
  • Risk: fire, theft or drive failure = data gone

Cloud

  • Access from any device, anywhere
  • Automatic off-site backup copy
  • Easy to share and collaborate
  • Risk: needs internet; ongoing subscription cost

Choosing Storage for the Job

Photographer

Huge RAW files, needs fast transfer and lots of capacity.

Fast NVMe SSD + cloud backup

School

Many users, shared access, needs reliability and easy recovery.

Cloud storage (Google/Microsoft)

Gamer

Large game installs, needs quick load times, budget-aware.

SSD for games, HDD for bulk storage

Test Your Knowledge!

Let's see what you've learned about storage

Q1: Which storage type has no moving parts?

AHDD
BSSD
CBlu-ray disc
DFloppy disk
Answer: B — SSDs use flash memory chips instead of spinning platters, making them faster and shock resistant.

Q2: Which storage type is usually cheapest per gigabyte for bulk data?

AHDD
BNVMe SSD
CPremium cloud tier
DHigh-end USB stick
Answer: A — HDDs remain the cheapest option per GB for large amounts of data.

Q3: What's a key risk of relying only on cloud storage?

AIt never runs out of space
BNo internet access means no access to your files
CIt can't be shared with anyone
DIt's always faster than an SSD
Answer: B — Cloud storage depends on a working internet connection, so it should complement, not replace, local backups.

Key Takeaways

Speed varies hugely

NVMe SSD > SATA SSD > HDD > network link

Cost trade-offs

HDD wins on £/GB, SSD wins on speed

Durability matters

No moving parts = fewer drops-kill-it risks

Mix & match

Most real setups combine several types

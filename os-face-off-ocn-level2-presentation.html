What Does an OS Actually Do?

The operating system is the software that manages everything else — hardware, apps, files and users — so you never have to talk to the hardware directly.

Manages Hardware

Shares CPU, RAM and devices between programs

Runs Software

Loads and controls every app you open

Manages Files

Organises data into a file system

Handles Users

Logins, permissions and security

What is Encryption?

Definition

Encryption scrambles readable data (plaintext) into unreadable data (ciphertext) using a mathematical process and a key — only someone with the right key can reverse it.

Think of it like this...

It's a locked box. Anyone can see the box, but only someone with the matching key can open it and read what's inside.

The Caesar Cipher

One of the oldest ciphers: shift every letter a fixed number of places in the alphabet.

Plaintext: H E L L O

Shift by 3 →

Ciphertext: K H O O R

Why It's Not Used Today

Only 25 possible shifts exist — a computer (or a patient human) can try them all in seconds. Real encryption uses far larger, more complex keys.

Symmetric vs Public-Key Encryption

Symmetric

  • Same key encrypts and decrypts
  • Fast — good for large amounts of data
  • Both sides must share the key securely first
  • Example: AES

Public-Key (Asymmetric)

  • A public key encrypts, a private key decrypts
  • Public key can be shared openly
  • Solves the "sharing the key safely" problem
  • Example: RSA — used to set up HTTPS connections

What the Padlock Confirms

  • Traffic between you and the site is encrypted in transit
  • Data can't easily be read or altered by someone in between

Stay Safe

Only send made-up practice messages — this is a paper-and-browser exercise, not real secure communication.

Q1: In a Caesar cipher, what does "shift by 3" mean?

ADelete every 3rd letter
BMove each letter 3 places forward in the alphabet
CRepeat the message 3 times
DConvert letters to numbers
Answer: B - Each letter is replaced with the one a fixed number of places later in the alphabet.

Q2: In public-key encryption, what can be shared openly?

AThe private key
BThe public key
CNeither key
DBoth keys equally in secret
Answer: B - The public key can be shared freely; only the matching private key can decrypt.

Q3: What does the padlock icon in a browser confirm?

AThe website is definitely trustworthy
BThe connection to the site is encrypted
CThe site has no adverts
DYour antivirus is up to date
Answer: B - HTTPS encrypts the connection; it doesn't guarantee the site's content is genuine.

Key Takeaways

Old Ciphers Are Weak

Caesar cipher teaches the idea; modern ciphers are far stronger

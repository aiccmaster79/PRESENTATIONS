What is a Network?

Definition

A network connects two or more devices so they can share data, resources and services—such as files, printers, internet access and applications.

Think of it like this...

Building a network is like designing a road system: devices are buildings, cables/Wi‑Fi are roads, switches are junctions, and routers are motorway gateways to other towns.

IP Addressing Basics

Every device needs an IP address to be found on the network—like a house number and street.

  • IPv4 example: 192.168.1.20
  • Subnet mask: defines which addresses are “local”
  • Default gateway: the router used to reach other networks
  • DNS: turns names (google.com) into IP addresses
  • DHCP: automatically hands out addresses

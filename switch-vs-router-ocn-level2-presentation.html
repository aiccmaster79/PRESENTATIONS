Switch vs Router

How switches and routers play different network roles

OCN NI Level 2 — Intro to networking

Learning Objectives

By the end of this session, you will be able to:

  • Describe what a switch does inside a LAN
  • Describe what a router does between networks
  • Identify typical home/lab device roles (router, switch, AP)
  • Trace a simple path from PC → switch → router → internet
  • Choose the right device when designing a small network

One-Line Difference

Switch: connects devices on the same network (LAN).
Router: connects different networks and chooses paths (often LAN ↔ Internet).

Tip

Many home “routers” are combo boxes: router + switch + Wi‑Fi AP.

Side by Side

Switch

  • Forwards frames using MAC addresses
  • Creates/extends a LAN
  • Many ports for PCs/printers
  • Usually no WAN port
  • Keeps local traffic local

Router

  • Forwards packets using IP addresses
  • Links networks / subnets
  • Often has WAN + LAN ports
  • Can do NAT, DHCP, firewall
  • Default gateway for clients

Demo Sequence

  1. Show two PCs plugged into a switch — ping each other (same LAN)
  2. Identify the default gateway IP on a client (ipconfig / ip route)
  3. Trace that gateway to the router’s LAN interface
  4. Disconnect WAN and show internet fails while LAN ping still works
  5. Discuss combo home gateway vs separate switch + router
  6. Sketch a small-office diagram together
